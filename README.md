## 📖 About the Project
A highly secure, encrypted vault application for managing sensitive credentials and documents. This project demonstrates advanced cryptography, secure key storage, and zero-knowledge architecture principles.

## ✨ Key Features
- 🛡️ **E2E Encryption**: Military-grade end-to-end encryption for all stored data.\n- 🔑 **Zero-Knowledge**: Architecture ensuring the server never knows the encryption keys.\n- 🎨 **Modern UI**: Clean, intuitive user interface focusing on security and privacy.
